Montreal's Restaurant Sanitary Violation Tracker

Description

This Flask-based web application tracks and displays sanitary violations in Montreal restaurants. It features a user-friendly search interface where users can find violations by establishment name, address, or owner, as well as filter results by date range. The app is backed by an SQLite3 database that stores violation records, and the data is synchronized daily to ensure accuracy. A fully documented API is also included, allowing developers to access and integrate the data into other systems.
