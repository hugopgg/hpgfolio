This project is a job alert application developed in C#, designed specifically to help me land my first job. Montreal's Restaurant Sanitary Violation Tracker

This web application, built using Flask, allows users to search for sanitary violations in restaurants across Montreal. Users can look up violations by establishment name, address, or owner, and also filter violations that occurred within a specific date range. In addition to the web interface, the application includes a fully documented API, enabling external users to access and integrate the data.

Mini Java Compiler

This Java-based project is a mini compiler that reads and interprets files containing code in a simple, predefined format. It provides functionality to process and analyze the code, including interpreting it into Java code. The project showcases basic compiler principles and text parsing techniques, offering a hands-on example of how compilers transform code into executable instructions.

Lazy copy in C

This simple C utility offers efficient file copying. By utilizing a lazy copying technique, it ensures that only the necessary data blocks are transferred, minimizing redundant data transfers. This makes the utility particularly suitable for large files and backups, allowing users to save time and storage space. Users can also specify the size of the data blocks to process, offering greater control over performance. With its straightforward approach, the utility provides an effective solution for managing file copies with ease.
