I'm a graduating student in Computer Science and Software Engineering at UQAM. This site features a few of the projects I worked on during my studies, focusing on different areas of software development.

Montreal's Restaurant Sanitary Violation Tracker

This web application, built using Flask, allows users to search for sanitary violations in restaurants across Montreal. Users can look up violations by establishment name, address, or owner, and also filter violations that occurred within a specific date range. In addition to the web interface, the application includes a fully documented API, enabling external users to access and integrate the data programmatically. This project highlights my skills in backend development, database management, and API design, as well as my ability to create a user-friendly and informative web application.
