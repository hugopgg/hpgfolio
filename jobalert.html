Job Alert C# Application

Description

This job alert application developed in C# facilitate the process of finding relevant job postings. It performs web scraping on multiple job sites to collect the latest listings, ensuring access to up-to-date opportunities. The application runs automatically every 24 hours, gathering new job postings and sending email notifications when fresh listings are available. Designed for efficiency, the application simplifies the task of monitoring job opportunities without the need to manually check various sites. The provided job sites and XPaths are for reference purposes. It can be easily deployed using Docker, making it adaptable for different environments. This application serves as a practical solution for automating my job search process.
